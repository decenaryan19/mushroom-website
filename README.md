# 🍄 Suwannee Hill Farms - Mushroom Landing Page

Isang modernong static website landing page na ginawa para sa isang organikong sakahan ng kabute. Naglalaman ito ng mga impormasyon tungkol sa "Organic Oak Log Forest Grown Shiitake Mushrooms," mga recipe, farm gallery, at mga artikulo.

Gisunod at ginaya ang disenyo mula sa orihinal na website gamit ang modernong AI developer tools.

## 🚀 Mga Features ng Website
- **Responsive Layout:** Swak at maganda tingnan sa Desktop, Tablet, at Mobile devices.
- **Modern Tech Stack:** Binuo gamit ang purong HTML, Tailwind CSS (via CDN), at kaunting JavaScript.
- **Dynamic Content Sections:**
  - Hero banner na may mga sariwang kabute.
  - Multi-column Product Highlight na may 'Buy Now' action.
  - Recipe grid na nagtatampok ng iba't ibang lutong kabute.
  - Interactive Farm Gallery at Articles section.
  - Kumpletong Footer na may Quick Links at Contact Info.

## 🛠️ Tech Stack
- **HTML5** - Para sa istraktura ng website.
- **Tailwind CSS** - Para sa mabilis at malinis na disenyo/styling.
- **Cursor AI** - Ginamit bilang pangunahing editor at AI coding assistant.
- **Cloudflare Pages** - Ginamit para sa mabilis at libreng web hosting.

## 📂 Istruktura ng Proyekto
- `index.html` - Ang pangunahing landing page / homepage.
- `about.html` - Pahina tungkol sa farm.
- `product.html` - Detalye ng mga produkto.
- `articles.html` - Blog at mga artikulo.
- `contact.html` - Contact form at lokasyon.
- `styles.css` - Custom styles para sa Tailwind layout.
- `script.js` - Simple scripts kung kinakailangan.
- `/images` - Folder para sa mga lokal na larawan at assets.

## 💻 Paano Patakbuhin sa Lokal na Computer
1. I-clone o i-download ang repository na ito.
2. Buksan ang folder sa iyong paboritong code editor (gaya ng VS Code o Cursor).
3. I-double click o buksan ang `index.html` gamit ang anumang web browser (Chrome, Brave, Safari).
