# 🍄 Suwannee Hill Farms - Mushroom Landing Page

A modern, responsive single-page static landing page created for an organic mushroom farm. This website showcases "Organic Oak Log Forest Grown Shiitake Mushrooms," recipes, a farm gallery, and informative articles.

The layout and design were carefully replicated from the original reference site using modern AI developer tools.

## 🚀 Features
- **Fully Responsive Layout:** Optimized and beautifully rendered across Desktop, Tablet, and Mobile devices.
- **Modern Tech Stack:** Built with pure HTML5, Tailwind CSS (via CDN), and lightweight JavaScript.
- **Dynamic Content Sections:**
  - High-quality hero banner slider featuring fresh mushrooms.
  - Multi-column Product Highlight with an eye-catching 'Buy Now' call-to-action.
  - Recipe grid showcasing various gourmet mushroom dishes.
  - Interactive Farm Gallery and dedicated Articles/Blog space.
  - Comprehensive multi-column Footer with Quick Links and Contact Information.

## 🛠️ Tech Stack
- **HTML5** - For semantic website structuring.
- **Tailwind CSS** - For rapid, clean utility-first styling and layout structure.
- **Cursor AI** - Utilized as the core code editor and AI assistant.
- **Cloudflare Pages** - Leveraged for fast, secure, and free global static hosting.

## 📂 Project Structure
- `index.html` - The main landing page / homepage.
- `about.html` - The company background and history page.
- `product.html` - Detailed inventory and product specifications.
- `articles.html` - Educational blog entries and resources.
- `contact.html` - Contact form, support details, and map location.
- `styles.css` - Custom style overrides alongside Tailwind utility classes.
- `script.js` - Simple structural scripts (e.g., handling navigation toggles).
- `/images` - Local assets folder for custom graphics and imagery.

## 💻 How to Run Locally
1. Clone or download this repository to your machine.
2. Open the project folder in your preferred code editor (such as VS Code or Cursor).
3. Double-click or open the `index.html` file using any modern web browser (Chrome, Brave, Firefox, Safari).
